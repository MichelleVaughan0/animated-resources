PROYECTO: SVG ANIMADO


Este proyecto incluye animaciones Lottie (JSON), las cuales requieren
un entorno HTTP para funcionar correctamente debido a políticas de seguridad
del navegador.

EJECUCIÓN LOCAL:
- Abrir la carpeta en Visual Studio Code
- Instalar la extensión Live Server
- Clic derecho sobre index.html
- Ejecutar "Open with Live Server"

VERSIÓN ONLINE:
https://michellevaughan0.github.io/animated-resources/
